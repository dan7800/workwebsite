Research Projects

This is a partial list of the research projects I am working on. Please contact me if you are interested in contributing.

  1. Darwin Project: A tool which downloads, analyzes and reports on several key analytics about Android apps including: Reported JLint errors, adherance to coding standards, utilized permissions, over permissions and under permissions. To date, the project has anayzed over 70,000 Android apps.
  2. AndroSec Project: Collects and analyzes Android version control repositories from F-Droid. In addition to various security and quality results gathered from static analysis tools, we've also collected version control information such as commit messages from the GIT repositories.
  3. Concolic Code Clone Detection: A tool I developed to detect software clones using concolic analysis.
