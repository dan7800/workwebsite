Installation Instructions

Option #1 - Install from VM (Recommended)

  1. Download VM
  2. You may want to make sure that the source code for the application is up to date. This may be done by running svn update . in the cccd_user directory.
  3. Run the application following the Instructions

Option #2 - Install from scratch.

Note: For each of the described components, specific installation instructions may be viewed on the component's website link included.

  1. Select a Unix environment for tool to be installed upon. Note: CCCD has only been tested on OSX 10.8.3 and Fedora 14 (64-bit)
  2. Install CREST
  3. Install CTAGS
  4. Install CCCD application
      5. The source code for the project should be retrieved by using the command svn checkout http://dk-crest-java.googlecode.com/svn/trunk/ dk-crest-java-read-only Further information may be found in the application's SVN repository.
  5. Install custom cil.ml file from the SVN. This file is located a bashScripts/customfiles/cil.ml and should be copied to /cil/src/

Running CCCD

Once CCCD has been properly installed, running the application is a simple process

  1. Move the source code of the target application to sourceFiles/input/"appname"
  2. Run CCCD by using the command ./cccd "appName" Note: The appName should exactly match the name of the folder file in the previous step.
  3. Wait for analysis process to conclude.
  4. View the final report information located in reports/"appname".csv
* If you have any problems, you may find the "HelpfulCommands.txt" document helpful.
